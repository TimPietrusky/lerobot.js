# LeRobot.js

> interact with your robot in JS, inspired by [LeRobot](https://github.com/huggingface/lerobot)

## Install

```bash
# Web library
npm install @lerobot/web

# Node.js library
npm install @lerobot/node

# CLI / terminal
npm install -g lerobot
# or
npx lerobot --help
```
